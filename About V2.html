The Project

This website was my first foray into web development using HTML, CSS and JS. The style sheet used on the 4 pages came from W3 Schools. The index page consists of a mix of text, picture boxes and an interactable map made using leaflet. The game itself is a web version of Simon Says, where the user must memorise a series of button prompts and recall them in the correct order. If the User guesses correctly the hue lights in the lab flash green, otherwise, the lights flash red instead. When a new round begins the lights flash orange. To help the user understand the purpose of the lights, a key page is also included that shows the values of the 3 colours.
